## Project setup

Public and Private key generated from https://cryptotools.net/rsagen
and save to 
- public_key.pem
- private_key.pem

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## API Documentation

Swagger documentation available at: http://localhost:3000/api-docs

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
